# Tribu Performance

Panel de administración para **Tribu Performance**, construido con **Next.js** y **Tailwind CSS**.

## Tecnologías

* Next.js 16.x
* React 19
* TypeScript
* Tailwind CSS v4

## Requisitos

* Node.js 18.x o superior (recomendado 20.x)

## Instalación

1. Clonar el repositorio e instalar dependencias:

   ```bash
   npm install
   # o
   yarn install
   ```

   > Si aparece un error de peer-dependencies, usar: `npm install --legacy-peer-deps`

2. Iniciar el servidor de desarrollo:

   ```bash
   npm run dev
   # o
   yarn dev
   ```

3. Abrir [http://localhost:3000](http://localhost:3000) en el navegador.
